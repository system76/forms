# Select

This is a super basic style only component. It has no special functionality
to the standard html `select` tag, except for the style.

## Props

<Props of={SysSelect} />

### Normal

This is how the select component looks without any given props or data.

<Preview>
  <Story height="100px" name="Normal">
    {{
      components: { SysSelect },
      template: `
        <sys-select>
          <option value="William Adama">William Adama</option>
          <option value="Laura Roslin">Laura Roslin</option>
          <option value="Kara Thrace">Kara Thrace</option>
        </sys-select>
      `
    }}
  </Story>
</Preview>
