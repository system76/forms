# Select

This is a form styled select box. it includes validation, error handling, and
everything you would expect from a high level component.

## Props

<Props of={SysFormSelect} />

### Normal

This is how the select component looks without any given props or data.

<Preview>
  <Story height="200px" name="Normal">
    {{
      template: `

### With Options

Instead of using html to template your options, you can specify them with the
`options` and `value` prop.

<Preview>
  <Story height="100px" name="Options">
    {{
      data: () => ({
        options: [
          ['CA', 'Canada'],
          ['MX', 'Mexico'],
          ['US', 'United States']
        ],
        value: 'US'
      }),
      template: `
        <sys-form-select
          v-model="value"
          id="options"
          label="Options"
          :options="options"
        />
      `
    }}
  </Story>
</Preview>
